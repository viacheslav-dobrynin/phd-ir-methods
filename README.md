# Методы информационного поиска

Данный репозиторий содержит реализации и эксперименты для методов, предложенных в кандидатской диссертации на тему: «Методы построения индексов для векторных баз данных с использованием глубокого обучения».

## Структура проекта

- [common](./common/) – общие файлы для обоих методов и скрипты для оценки существующих алгоритмов
- [infra](./infra/) – Terraform-конфигурация инфраструктуры, используемой для оценки предложенных методов
- [inverted-index](./inverted-index/) – реализация собственного запроса для Lucene
- [spar_k_means_bert](./spar_k_means_bert/) – первый метод, соответствующий первому положению, выносимому на защиту
- [sparsifier_model](./sparsifier_model/) – второй метод, соответствующий второму положению, выносимому на защиту
- [tools](./tools/) – полезные инструменты для экспериментов: сборка PyLucene, установка зависимостей и т. д.
